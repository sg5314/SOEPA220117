# Installation & Usage

### 1.プロジェクトをクローン

    git clone https://github.com/sg5314/SOEPA220117.git

### 2.Dockerイメージの作成

    docker-compose build

### 3.Dockerコンテナの起動

    docker-compose up

### 4.以下のURLにアクセス

* http://127.0.0.1:3000/


## Dockerコンテナの削除

    docker-compose down
